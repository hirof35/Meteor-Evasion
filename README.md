🚀 Meteor Evasion (隕石サバイバル)Canvas API と Node.js を利用した、シンプルながら中毒性のある宇宙船アクションゲームです。迫りくる隕石を避け続け、ハイスコアを目指しましょう！🌟 特徴滑らかな操作感: 慣性を利用したスムーズな移動と、機体の傾き演出。動的な演出: 隕石がランダムな速度で回転しながら落下。難易度選択: EASY, NORMAL, HARD の3段階から自分に合ったレベルを選択可能。精密な当たり判定: 見た目の違和感を減らすため、ヒットボックスを最適化。🛠 セットアップと実行方法ファイルの配置同じフォルダ内に以下のファイルを配置してください。index.html: ゲーム画面game.js: ゲームロジックplayer.png: 宇宙船の画像meteor.png: 隕石の画像実行
Node.js がインストールされている環境で、コマンドプロンプトやターミナルを開き以下のコマンドを実行します:  DOSnpx serve .
